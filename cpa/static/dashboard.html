Enroll Mailboxes

Enroll via Graph API

Requires Azure AD config

Enter a user's email address. The system will pull their sent-mail history from Microsoft Graph and build a behavioral profile.

Enroll via Direct Upload

Always available

POST emails directly to the /enroll endpoint. Use this if Graph isn't configured or you're ingesting from another source. 

curl -X POST http://localhost:8000/enroll \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_KEY" \
  -d '{
    "tenant_id": "default",
    "user_id": "sha256-prefix-of-email",
    "user_email": "user@example.com",
    "emails": [
      {"recipient_email": "bob@example.com", "body": "Email text...", "sent_at": "2024-01-15T09:00:00Z"}
    ]
  }'
